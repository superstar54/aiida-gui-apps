
Welcome to AiiDA GUI apps
====================================================
The AiiDA GUI apps are a set of web applications that provide a user-friendly interface for running and managing AiiDA workflows.



.. note::

   If you encounter any problems, please update the widget to the latest version. If the problem persists, please report it on the `GitHub issue <https://github.com/superstar54/aiida-gui-apps/issues>`_
